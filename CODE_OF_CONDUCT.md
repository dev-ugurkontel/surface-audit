# Code of Conduct

`surface-audit` is a solo-maintained project. The maintainer expects
**minimal-friction, professional interactions** in every public space
tied to this repository — issues, pull requests, Discussions, and
anywhere else the project is referenced.

## Expected

- Be concise and factual in reports and proposals.
- Assume good faith until the evidence says otherwise.
- Respect the maintainer's judgment; this project is not run by
  consensus.

## Not tolerated

- Personal attacks, harassment, or discrimination of any kind.
- Misrepresenting the project, the maintainer, or other participants.
- Spam or off-topic self-promotion.

## Enforcement

The maintainer may, at any time and without prior notice:

- edit or remove any content tied to this project,
- lock or close any issue, PR, or discussion,
- block any user from the repository.

Report conduct issues privately via the email address listed on the
maintainer's GitHub profile. Reports are handled confidentially.

The project aligns with the spirit of the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
but the maintainer's decisions are final.
